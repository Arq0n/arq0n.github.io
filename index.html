Productivi-Day

Productivi-Day is a web application designed to help the user eliminate any potential bad habits they may have in their schedule. The application processes the schedule using machine learning APIs to suggest an improved schedule as well as a summary of what has changed. The web app utilizes React and Tailwind CSS for the frontend and Node.js for the backend.
