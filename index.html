Projects

Productivi-Day is a web application designed to help the user eliminate any potential bad habits they may have in their schedule. The application processes the schedule using machine learning and APIs to generate an improved schedule as well as a summary of what has changed. Utilized React and Tailwind CSS to create the frontend and Node.js for the backend.
